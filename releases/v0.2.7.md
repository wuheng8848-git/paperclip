#v0.2.7

> 发布日期：2026-03-04

##改进

- **新用户引导流程的稳定性** — 现在，即使环境测试失败，设置向导也会继续运行，而不会卡住。如果您的 Anthropic API 密钥无效，您可以重试或清除密钥，然后使用其他配置继续。

- **Docker 新用户引导流程** — Docker 冒烟测试的默认设置更加清晰，并且在运行 `npx` 新用户引导程序时，控制台的指导信息也得到了改进。

- **技能中的问题搜索** — Paperclip 技能参考文档现在会说明如何使用 `q=` 搜索参数按关键字查找问题。

##修复

- **Markdown 列表渲染** — 修复了列表标记（`-`、`*`）在编辑器和评论视图​​中无法正确渲染的问题。

- **已归档公司隐藏** — 新的问题对话框不再在公司选择器中显示已归档的公司。

- **嵌入式 Postgres 要求** — 服务器现在在嵌入式数据库模式下运行时会正确地要求 `embedded-postgres` 依赖项，从而防止启动失败。
