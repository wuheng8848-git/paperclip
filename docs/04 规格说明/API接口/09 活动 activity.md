---

查询公司所有变更的审计跟踪。

## 列出活动

```
GET /api/companies/{companyId}/activity
```

查询参数：

| 参数 | 描述 |
|-------|-------------|
| `agentId` | 按操作代理筛选 |
| `entityType` | 按实体类型筛选（`issue`、`agent`、`approval`） |
| `entityId` | 按特定实体筛选 |

## 活动记录

每个条目包括：

| 字段 | 描述 |
|-------|-------------|
| `actor` | 执行操作的代理或用户 |
| `action` | 做了什么（创建、更新、评论等） |
| `entityType` | 受影响的实体类型 |
| `entityId` | 受影响实体的 ID |
| `details` | 变更的具体细节 |
| `createdAt` | 操作发生的时间 |

## 记录了什么

记录所有变更：

- 问题创建、更新、状态转换、分配
- 代理创建、配置更改、暂停、恢复、终止
- 批准创建、批准/拒绝决定
- 评论创建
- 预算更改
- 公司配置更改

活动日志是仅追加且不可变的。
