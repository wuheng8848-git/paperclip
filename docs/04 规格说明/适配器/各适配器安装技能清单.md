title: 各适配器安装技能清单
summary: 公司技能库导入与按适配器落地的速查（本机目录与专文索引）
---

「装技能」在本仓库里通常有**两层**：

1. **进公司**：在 Board **公司技能库**里导入/更新（来源常为仓库 `skills/` 或自定义包）。
2. **给智能体用**：在智能体上勾选 `desiredSkills`；运行前由各适配器把正文落到**对应 CLI 认的目录**（多为符号链接/复制），工具才能加载 `SKILL.md`。

下列为**内置适配器**的常见落点速查（`$HOME` 可被 `adapterConfig.env.HOME` 改写；远程/沙箱另有运行时路径，见各适配器专文）。**看板里「已应用技能」的文案（如「保留在工作区」）以当前实例/UI 为准。**

| 适配器（文中名） | 类型键 | 技能落点（典型） | 专文 / 说明 |
|------------------|--------|------------------|-------------|
| Claude 本地 | `claude_local` | `$HOME/.claude/skills` | [07 Claude 本地适配器](07%20Claude%20本地适配器%20claude-local.md) |
| Codex 本地 | `codex_local` | `$CODEX_HOME/skills`（未设时多为 `$HOME/.codex/skills`） | [08 Codex 本地适配器](08%20Codex%20本地适配器%20codex-local.md) |
| Cursor 本地 | `cursor` | `$HOME/.cursor/skills` | [12 Cursor 本地适配器](12%20Cursor%20本地适配器%20cursor-local.md)（物化/必选要求见专文） |
| Gemini 本地 | `gemini_local` | `$HOME/.gemini/skills` | [09 Gemini 本地适配器](09%20Gemini%20本地适配器%20gemini-local.md) |
| CodeBuddy 本地 | `codebuddy_local` | `$HOME/.codebuddy/skills` | [10 CodeBuddy 本地适配器](10%20CodeBuddy%20本地适配器%20codebuddy-local.md) |
| Qwen 本地 | `qwen_local` | `$HOME/.qwen/skills` | [11 Qwen 本地适配器](11%20Qwen%20本地适配器%20qwen-local.md) |

| HTTP | `http` | — | [05 HTTP 适配器](05%20HTTP%20适配器%20http.md) |

| ACP X / Hermes / OpenClaw / Grok 等 | `acpx_local` / `hermes_local` / `openclaw_gateway` / `grok_local` 等 | 以各包 `skills.ts` 与专文为准 | [01 适配器概览](01%20适配器概览%20overview.md) |

**技能包正文真源**：仓库 `skills/*/SKILL.md`（及公司库副本）；总览见仓库根 `skills/README.md`。

**本机自己往工具目录里扔的额外技能夹**：看板可标成「用户自行安装」，与公司库勾选**并存**，数目也会分开显示。
