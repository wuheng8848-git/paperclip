# 成本页：英文 UI 与中文化建议

## 范围

- **场景**：`/costs`（公司前缀路由下同页，如 `/ROU/costs`）、本页使用的日期预设、概览/预算/供应商/计费方/账务 Tab 及其子卡片与空状态。
- **排除**（本单不对照；或另行事务）：
  - **`BudgetPolicyCard`**：已走 `budgetPolicyUi` / `formatBudgetScopeTypeZh` 等，以现有中文为准。
  - **原始标识**：`provider`/`biller`/`model` slug、`providerDisplayName` 中已固化的英文品牌名（Anthropic、OpenAI 等）、API/后端返回的 `quotaError`、`FinanceEvent.description`、`externalInvoiceId` 等自由文本。
  - **`window.label` 等上游透传字符串**（Claude/Codex 订阅额度窗格）：仅标注「可择期映射或保留英文」。
- **附录**：命令面板中硬编码 **`Costs`**（与侧栏 `nav.costs` 已「成本」可对齐）。
