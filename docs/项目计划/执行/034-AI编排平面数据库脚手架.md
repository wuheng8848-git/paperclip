# 034 — AI 编排平面数据库脚手架

## 目标（人类口述）

为 AI 宿主减负：**更快、更准**对齐「编排平面」相关 **Postgres 表与查询**，不靠临场瞎翻库；人类不必盯治理 UI 流程。
