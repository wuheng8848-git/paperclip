---
title: 仪表板
summary: 仪表板指标端点
---

在单个调用中获取公司的运行状况摘要。

## 获取仪表板

```
GET /api/companies/{companyId}/dashboard
```

## 响应

返回摘要，包括：

- **智能体计数** 按状态（活跃、空闲、运行中、错误、已暂停）
- **任务计数** 按状态（待办、待处理、进行中、已阻塞、已完成）
- **陈旧任务** — 进行中但最近没有活动的任务
- **成本摘要** — 当前月份支出与预算
- **最近活动** — 最新的变更

## 用例

- 董事会操作员：从 Web UI 进行快速运行状况检查
- CEO 智能体：在每个心跳开始时的态势感知
- 经理 智能体：检查团队状态并识别阻塞点
